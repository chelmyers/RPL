Welcome

This is a responsive pattern library. You can use it as a reference for common responsive pattern. This library can also be used as a template to create your own. You can download it at: https://github.com/chelmyers/RPL. These pages are static and created with HTML and CSS. If you are a more advanced web developer, feel free to modify these templates and fit them into your current work flow.

This library was created for the book Learning Responsive Web Design Patterns by Chelsea Myers. Check out more books at Packt Publishing and follow Chelsea at @chelmyers.
