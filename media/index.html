Picture ElementView Example

A responsive image solution using picture, source, and img elements.

Example

HTML


<picture>
	<source 
	  media="(min-width: 1000px)"
	  srcset="i/1920.png">
	<source 
	  media="(min-width: 500px)"
	  srcset="i/1000.png">
	<img 
	 src="i/500.png" 
	 alt="Featured Image">
</picture>

CSS


img {
  max-width: 100%;
}

Full-Span ImageView Pattern

Easy

  • Pros:

  • Easy to implement
  • Clear and accessible

  • Cons:

  • Long navigations can take up too much screen realestate on smaller screens
  • Not scalable

Example

HTML


<header>
   <img id="banner-img" src="i/banner-1400.png" srcset="i/banner-2000.png 2x"  sizes="100vw" alt="Banner Image">
   <div id="banner-overlay">
     <h1>Company Name</h1>
     <p>A very catchy and cool tagline.</p>
   </div>
</header>

CSS


header {
  position: relative;
}

  #banner-img {
  width: 100%;
}

#banner-overlay {
  position: absolute;
  text-align: center;
  top:50%; left:50%;
  transform: translate(-50%, -50%);
}
