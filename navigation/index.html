Navigation

  • Patterns:
  • Simple Stack
  • Anchor & Footer
  • The Toggle
  • Off Canvas

A website's navigation is the most important tool to the user. It should provide quick access to all the website has to offer. It should also not get in the way. Below are examples of popular responsive patterns that make navigation websites easy on both small and large screens.

Simple StackView Pattern

Easy This pattern is very simple and easy to implement. The navigation does not change much from small to large screen. All pages can be seen as once and sometimes shift under the page title on smaller screens.

  • Pros:

  • Easy to implement
  • Clear and accessible

  • Cons:

  • Long navigations can take up too much screen realestate on smaller screens
  • Not scalable

Example

HTML


<header>
	<h1>Page Heading (h1)</h1>
	<nav>
		<ul>
			<li><a href="#">Home</a></li>
			<li><a href="#">About</a></li>
			<li><a href="#">Our Portfolio</a></li>
			<li><a href="#">Our Services</a></li>
			<li><a href="#">Contact Us</a></li>
		</ul>
	</nav>
</header>

CSS


header {
	overflow: auto;
}

header h1 {
	margin-bottom: 0.25em;

}

nav li {
	list-style: none;
	padding: 0.25em 0.5em 0 0;
}
	nav li a {
		text-decoration: none;
	}


@media only screen and (min-width: 750px){
	header h1 {
		float: left;
		margin:0;
	}

	nav {
		float: right;
	}
		nav li {
			display: inline-block;
		}
}

Anchor & Footer View Pattern

Easy This pattern consists of a jump link in the header that links to a menu in the Anchor. It is easy to create but the jump to the bottom of the page can be jarring.

  • Pros:

  • Easy to implement
  • Can fit large menus
  • Saves screen realesta

  • Cons:

  • User won't think to look for the menu in the Anchor
  • Jumping to the bottom of the page can be jarring

Example

HTML


<header>	
		<img src="http://placehold.it/100x50&=logo" alt="Logo">
		<p class="menu-btn"><a href="#menu">Menu</a></p>
	</header>
	<div class="content">
		<p>Content goes here.</p>
	</div>
	<Anchor>
		<div class="Anchor-inner">
			<h3>Anchor</h3>
			<nav id="menu">
				<ul>
					<li><a href="#">Home</a></li>
					<li><a href="#">About</a></li>
					<li><a href="#">Our Portfolio</a></li>
					<li><a href="#">Our Services</a></li>
					<li><a href="#">Contact Us</a></li>
				</ul>
			</nav>
		</div>
	</Anchor>

CSS


header {
	overflow: auto;
}

.menu-btn {
	float:right;
}

nav li {
	list-style: none;
	padding: 0.50em;
}
	nav li a {
		text-decoration: none;
	}

footer {
	padding: 0.75em;
}

@media screen and (min-width: 750px) {
	.menu-btn {
		display: none;
	}

	nav {
		position: absolute;
		top:0.75em;
		right:0.75em;
	}
	nav li {
		display: inline-block;
	}
}

Example

HTML


<body id="page">
  <header>
    <div class="header-inner">   
      <img src="http://placehold.it/100x50&=logo" alt="Logo">
      <a href="#" id="menuBtn">Menu</a>
    </div>

    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Our Portfolio</a></li>
        <li><a href="#">Our Services</a></li>
        <li><a href="#">Contact Us</a></li>
      </ul>
    </nav>
  </header>

CSS


header {
  padding:0;
}

nav {
  padding: 0.75em;
  background: lightgray;
  display: none;
}

  nav li {
    padding: 0.25em 0 0 0.5em;
  }

  nav a {
    display: block;
  }

#menuBtn {
  float: right;
  text-decoration: none;
  padding: 0.5em;
  border: 1px solid lightgray;
  border-radius: 3px;
}

#menuBtn:hover {
  background-color: lightgray;
}

.open nav {
  display: block;
}
@media screen and (min-width: 750px) {
  #menuBtn {
    display: none;
  }

  nav {
    overflow: auto;
    display: block;
  }

  nav ul {
    float: right;
  }
  nav li {
    display: inline-block;
  }

}

JavaScript


 var menuBtn = document.getElementById("menuBtn");
    var page = document.getElementById("page");

    function toggleClass(el, classWanted) {
      var classes = el.className.split(/\s+/);
      var found;

      for(i = 0; i < classes.length; i++) {
        if (classes[i] === classWanted) {
          classes.splice(i, 1);
          var found = true;      }
      }

      if (found != true) {
          classes.push(classWanted);
      }

      el.className = classes.join(' ');
    }

    menuBtn.onclick = function() {
     toggleClass(page, "open");
    }

HTML


<body id="page">
  <div id="oncanvas">
    <header>
      <div class="header-inner">
        <img src="http://placehold.it/100x50&=logo" alt="Logo">
        <a href="#" id="menuBtn">Menu</a>
        <nav>
          <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <li><a href="#">Our Portfolio</a></li>
            <li><a href="#">Our Services</a></li>
            <li><a href="#">Contact Us</a></li>
          </ul>
      </nav>
      </div>
    </header>
  </div>

  <nav id="offcanvas">
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Our Portfolio</a></li>
      <li><a href="#">Our Services</a></li>
      <li><a href="#">Contact Us</a></li>
    </ul>
  </nav>
</body>

CSS


header {
  padding:0;
  width: 100%;
}

nav {
  padding: 0.75em;
}

  nav li {
    padding: 0.25em 0 0 0.5em;
  }

  nav a {
    display: block;
  }


#oncanvas {
  width: 100%;
}

  #oncanvas nav {
    display: none;
  }

#offcanvas {
  display: none;
  width: 200px;
  position: absolute;
  right:0;
  top:0;
  bottom:0;
  background: lightgray;
}

.open #offcanvas {
  display: block;
}

.open #oncanvas {
  position: absolute;
  left: -200px;
}

#menuBtn {
  float: right;
  text-decoration: none;
  padding: 0.5em;
  border: 1px solid lightgray;
  border-radius: 3px;
}

#menuBtn:hover {
  background-color: lightgray;
}

@media screen and (min-width: 750px) {
  #menuBtn {
    display: none;
  }

  #oncanvas nav {
    display: inline-block;
    float:right;
  }

  nav li {
    display: inline-block;
  }

}
