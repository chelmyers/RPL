HeaderView Pattern

This is a responsive pattern library template. You can download it at: something.io/repo. This is created uses HTML and CSS. More advance developers can adapt this template using Grunt or PHP.

Example

HTML


<header>
	<div class="header-inner">
		<h1>Company Name</h1>
		<nav>
			<ul>
				<li><a href="#">Item 1</a></li>
				<li><a href="#">Item 2</a></li>
				<li><a href="#">Item 3</a></li>
			</ul>
		</nav>
	</div>
</header>

CSS


header {
	width: 100%;
	overflow: auto;
}

.header-inner {
	max-width: 1330px;
	overflow: auto;
	margin:0 auto;
}

header h1 {
	display: inline-block;
}

nav, nav ul {
	margin:0;
	padding:0;
}

nav li {
	list-style: none;
	display: inline-block;
	padding: 0.25em;
}
	nav li a {
		text-decoration: none;
	}


@media only screen and (min-width: 500px){
	header h1 {
		float: left;
	}
	nav, nav ul {
		float: right;
		display: inline-block;
	}
}
