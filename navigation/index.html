Navigation

A website's navigation is the most important tool to the user. It should provide quick access to all the website has to offer. It should also not get in the way. Below are examples of popular responsive patterns that make navigation websites easy on both small and large screens.

Top Navigation or "Do Nothing" PatternView Pattern

Easy This pattern is very simple and easy to implement. The navigation does not change much from small to large screen. All pages can be seen as once and sometimes shift under the page title on smaller screens.

  • Pros:

  • Easy to implement
  • Clear and accessible

  • Cons:

  • Long navigations can take up too much screen realestate on smaller screens
  • Not scalable

Example

HTML


<header>
	<div class="header-inner">
		<h1>Company Name</h1>
		<nav>
			<ul>
				<li><a href="#">Item 1</a></li>
				<li><a href="#">Item 2</a></li>
				<li><a href="#">Item 3</a></li>
			</ul>
		</nav>
	</div>
</header>

CSS


header {
	width: 100%;
	overflow: auto;
}

.header-inner {
	max-width: 1330px;
	overflow: auto;
	margin:0 auto;
}

nav, nav ul {
	margin:0;
	padding:0;
}

nav li {
	list-style: none;
	display: inline-block;
	padding: 0.25em;
}
	nav li a {
		text-decoration: none;
	}


@media only screen and (min-width: 500px){
	header h1 {
		float: left;
		display: inline-block;
	}
	nav {
		float: right;
		text-align: right;
	}
}

Anchor to Footer Pattern View Pattern

Easy This pattern consists of a jump link in the header that links to a menu in the footer. It is easy to create but the jump to the bottom of the page can be jarring.

  • Pros:

  • Easy to implement
  • Can fit large menus
  • Saves screen realesta

  • Cons:

  • User won't think to look for the menu in the footer
  • Jumping to the bottom of the page can be jarring

Example

HTML


<header>
	<div class="header-inner">
		<h1>Header</h1>
		<p class="menu-btn"><a href="#menu">Menu</a></p>
	</div>
</header>
<div class="content">
	<p>Content goes here.</p>
</div>
<footer>
	<div class="footer-inner">
		<h3>Footer Menu</h3>
		<nav id="menu">
			<ul>
				<li><a href="#">Item 1</a></li>
				<li><a href="#">Item 2</a></li>
				<li><a href="#">Item 3</a></li>
			</ul>
		</nav>
	</div>
</footer>

CSS


header {
	width: 100%;
	overflow: auto;
}
	header h1 {
		display: inline-block;
	}

.header-inner, .footer-inner{
	max-width: 1330px;
	overflow: auto;
	margin:0 auto;
	padding: 1em;
}

.menu-btn {
	float:right;
}

nav, nav ul {
	margin:0;
	padding:0;
}

nav li {
	list-style: none;
	padding: 0.50em;
}
	nav li a {
		text-decoration: none;
	}
